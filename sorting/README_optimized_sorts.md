# optimized-sorting-algorithms

This project compares several sorting algorithms, including optimized versions. It measures runtime differences across random, reverse-sorted, and almost-sorted lists.

This project was completed for CSC506 â€“ Module 6.

## How It Works

1. Lists of various sizes are generated.
2. Bubble Sort, Merge Sort, and Quick Sort all run on each list.
3. Optimized versions include improvements like early exits and better pivot selection.
4. Runtimes are calculated and printed.

## Features

**Sorting Algorithms**
- Bubble Sort  
- Optimized Bubble Sort  
- Merge Sort  
- Optimized Merge Sort  
- Quick Sort  
- Optimized Quick Sort  

**Testing System**
- Multiple list sizes  
- Multiple list patterns  
- Printed timing comparisons
