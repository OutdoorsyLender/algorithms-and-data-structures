# nfl-jersey-linear-search

This is a simple Python program that demonstrates linear search using an NFL jersey example. The user types in a team or player, and the program scans the list to find matching jerseys.

This project was completed as part of my CSC506 coursework.

## How It Works

The program runs in the terminal and works like this:

1. A list of jersey dictionaries is created.
2. The user enters search text.
3. A linear search checks each jersey.
4. Matching jerseys are printed.
